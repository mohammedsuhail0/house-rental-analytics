# 🏠 House Rental Analytics Project

This project explores Hyderabad house rental data to understand how factors like locality, furnishing type, tenant preference, and area influence rental prices 📊

It combines data cleaning, exploratory analysis, machine learning, and dashboard development into one complete workflow.

## 📂 Project Files

* `01_Data_Cleaning.ipynb`
  Cleaning missing values, fixing structure, preparing usable dataset

* `02_Exploratory_Data_Analysis.ipynb`
  Exploring rent distribution, locality patterns, furnishing trends

* `03_Model_Training_and_Testing.ipynb`
  Training and evaluating a rent prediction model

* `Cleaned_House_Data.csv`
  Final cleaned dataset used across the project

* `housing_rental_analytics.pbix`
  Power BI dashboard built manually for visual analysis

## 🛠 Tools Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Power BI

## 📈 Key Insights

* Rental prices vary strongly by locality
* Furnishing has visible effect on monthly rent
* Tenant type influences pricing patterns
* Area vs rent relationship contains clustering and outliers

## 🚀 Project Goal

The aim was to move from raw housing data to useful visual insights and predictive understanding through a complete analytics workflow.
